# Music recommendation based on mood using Emotion Recognition

Music recommendation based on mood using Emotion Recognition OpenCV and DeepFace

## Requirements

- deepface==0.0.93
- numpy==1.26.4
- opencv-python==4.10.0.84
- pandas==2.2.3
- scikit-learn==1.5.2
- tf_keras==2.18.0


## Installation

```bash
pip install -r requirements.txt
```

## Running the program
### Song classification
```bash
python song.py
```
### Main application
```bash
python emotion.py
```
